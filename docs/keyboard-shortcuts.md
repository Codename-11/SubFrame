# Keyboard Shortcuts

SubFrame provides keyboard shortcuts for fast navigation and terminal management. Press <kbd>Ctrl+?</kbd> (<kbd>Ctrl+Shift+/</kbd>) at any time to open the in-app shortcuts overlay.

All shortcuts use <kbd>Ctrl</kbd> on Windows/Linux and <kbd>Cmd</kbd> (⌘) on macOS unless noted otherwise.

## General

| Shortcut | macOS | Action |
|----------|-------|--------|
| <kbd>Ctrl+,</kbd> | <kbd>⌘+,</kbd> | Open Settings |
| <kbd>Ctrl+?</kbd> | <kbd>⌘+?</kbd> | Open keyboard shortcuts help |
| <kbd>Escape</kbd> | <kbd>Escape</kbd> | Close overlay / navigate back in full-view |

## Sidebar

| Shortcut | macOS | Action |
|----------|-------|--------|
| <kbd>Ctrl+B</kbd> | <kbd>⌘+B</kbd> | Toggle sidebar (show/hide) |
| <kbd>Ctrl+E</kbd> | <kbd>⌘+E</kbd> | Focus project list in sidebar |
| <kbd>Ctrl+Shift+E</kbd> | <kbd>⌘+Shift+E</kbd> | Focus file tree in sidebar |

## Terminal

| Shortcut | macOS | Action |
|----------|-------|--------|
| <kbd>Ctrl+Shift+T</kbd> | <kbd>⌘+Shift+T</kbd> | Create new terminal |
| <kbd>Ctrl+Shift+W</kbd> | <kbd>⌘+Shift+W</kbd> | Close current terminal |
| <kbd>Ctrl+Tab</kbd> | <kbd>⌘+Tab</kbd> | Switch to next terminal |
| <kbd>Ctrl+Shift+Tab</kbd> | <kbd>⌘+Shift+Tab</kbd> | Switch to previous terminal |
| <kbd>Ctrl+1</kbd> through <kbd>Ctrl+9</kbd> | <kbd>⌘+1</kbd> through <kbd>⌘+9</kbd> | Jump to terminal by position |
| <kbd>Ctrl+Shift+G</kbd> | <kbd>⌘+Shift+G</kbd> | Toggle grid/tabs view |
| <kbd>Ctrl+Shift+Enter</kbd> | <kbd>⌘+Shift+Enter</kbd> | Start AI tool in new terminal |
| <kbd>Ctrl+Shift+D</kbd> | <kbd>⌘+Shift+D</kbd> | Pop out terminal to separate window |
| <kbd>Ctrl+C</kbd> | <kbd>⌘+C</kbd> | Copy selected text (when text is selected) |
| <kbd>Ctrl+V</kbd> | <kbd>⌘+V</kbd> | Paste from clipboard |

::: tip Double-click to rename
Double-click any terminal tab to rename it. Press <kbd>Enter</kbd> to confirm or <kbd>Escape</kbd> to cancel.
:::

|----------|-------|--------|
| <kbd>Ctrl+Shift+S</kbd> | <kbd>⌘+Shift+S</kbd> | Toggle Sub-Tasks panel |
| <kbd>Ctrl+Shift+A</kbd> | <kbd>⌘+Shift+A</kbd> | Toggle Agent Activity panel |
| <kbd>Ctrl+Shift+X</kbd> | <kbd>⌘+Shift+X</kbd> | Toggle Plugins panel |
| <kbd>Ctrl+Shift+H</kbd> | <kbd>⌘+Shift+H</kbd> | Toggle History panel |
| <kbd>Ctrl+Shift+G</kbd> | <kbd>⌘+Shift+G</kbd> | Toggle GitHub panel |
| <kbd>Ctrl+Shift+Y</kbd> | <kbd>⌘+Shift+Y</kbd> | Toggle Pipeline panel |
| <kbd>Ctrl+Shift+L</kbd> | <kbd>⌘+Shift+L</kbd> | Toggle Prompt Library panel |
| <kbd>Ctrl+/</kbd> | <kbd>⌘+/</kbd> | Open Command Palette |
| <kbd>Ctrl+Shift+P</kbd> | <kbd>⌘+Shift+P</kbd> | Open Command Palette (alternate) |

::: info Shared shortcut
<kbd>Ctrl+Shift+G</kbd> is mapped to both the grid view toggle and the GitHub panel. The grid view toggle takes priority when a terminal is active.
:::

## Full-View Overlays

| Shortcut | macOS | Action |
|----------|-------|--------|
| <kbd>Ctrl+Shift+O</kbd> | <kbd>⌘+Shift+O</kbd> | Toggle project overview (full-view) |

## Editor

These shortcuts are available when the file editor is open.

| Shortcut | macOS | Action |
|----------|-------|--------|
| <kbd>Ctrl+S</kbd> | <kbd>⌘+S</kbd> | Save file |
| <kbd>F11</kbd> | <kbd>F11</kbd> | Toggle editor fullscreen |

## File Tree

When the file tree panel is focused, use these keys to navigate:

| Key | Action |
|-----|--------|
| <kbd>Arrow Up</kbd> / <kbd>Arrow Down</kbd> | Move focus between files |
| <kbd>Arrow Right</kbd> | Expand focused directory |
| <kbd>Arrow Left</kbd> | Collapse focused directory |
| <kbd>Enter</kbd> | Open focused file in editor |

## Tips

- The in-app shortcuts overlay (<kbd>Ctrl+?</kbd>) provides a quick visual reference without leaving your workflow.
- Tooltips on toolbar buttons show their keyboard shortcuts -- hover over any button to discover its shortcut.
- Terminal tabs can be reordered and renamed via the right-click context menu.
- Mouse back button navigates back in full-view overlays, matching the <kbd>Escape</kbd> behavior.
