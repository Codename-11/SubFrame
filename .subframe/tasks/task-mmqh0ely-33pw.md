Add an opt-in setting to make the ActivityBar always visible as a VS Code-style status bar. When enabled, shows git branch, active AI tool, usage stats, and activity streams in a persistent 24px bottom bar. When disabled (default), ActivityBar only appears when activity streams exist. This consolidates scattered status info (usage pill in tab bar, AI tool in settings) into one surface and provides a natural extension point for future integrations/plugins.
